## 📌 깃 커밋 컨벤션
```
Add : 새로운 파일 추가
Feat : 새로운 기능 추가
Fix : 버그 수정
Docs : 문서 수정
Style : 코드 formatting, 세미콜론(;) 누락, 코드 변경이 없는 경우
Design : css 수정
Rename : 파일 및 폴더 구조 변경
Refactor : 코드 리팩토링
Test : 테스트 코드, 리팽토링 테스트 코드 추가
Chore : 빌드 업무 수정, 패키지 매니저 수정
Remove : 파일 삭제
Modify : 코드 단순 수정
```
